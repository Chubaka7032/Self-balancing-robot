# Self-balancing-robot
суть проєкта полягає в тому, щоб змусити робота на двох колесах тримати баланс та не впасти.
Модулі робота: 
- Arduino uno
- L298N motor driver
- MPU 6050 sensor  (гіроскоп акселометр)
- Motors with wheels
Головне управління надходить з плати управління Arduino, вона зчитує данні з гіроскопа, що визначає нахил платформи до якої він прикріпленний,
 та управляючи моторами намагається утримувати баланс.Живлення надає пара батарейок типу АА.
 Робот має вигляд башні з колесами по боках внизу ,кріпляться до першої платформи на якій закріплені батарейки та плата управління
 по кутах цієї ж платформи є стрижні ,які кріплять верхню платформу , під верхньою закірпоений гіроскоп , все зєднано звчиайними проводами.
